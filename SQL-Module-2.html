Module 2: Data Retrieval & Filtering

The SELECT Statement

The SELECT statement is used to fetch data from a database. It is the most common SQL command.

SELECT column1, column2 FROM table_name;

Selecting All Columns

Use the asterisk (*) to select all available columns in a table.

SELECT * FROM Employees;

The WHERE Clause

The WHERE clause filters records. It ensures that only data fulfilling a specific condition is retrieved.

SELECT Name, Salary FROM Employees WHERE Salary > 50000;

Comparison Operators

  • = : Equal to
  • <> or != : Not equal to
  • <, > : Less than, Greater than
  • <=, >= : Less than or equal to, Greater than or equal to
