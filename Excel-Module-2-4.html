2.4 Strategic Logic & Functions

Mean, Variance, and Decision Making

The Strategic Logic:

A manager's primary role is to maintain the "Mean" (Average) and minimize "Variance" (Risk).

1. Statistical Functions

  • Central Tendency: Using SUM, AVERAGE, and MEDIAN to identify the "Boutique Pulse."
  • The Volatility Check: Utilizing STDEV.P to measure sales consistency.
    • High Std Dev: Indicates unpredictable footfall; requires higher "Safety Stock" levels.
    • Low Std Dev: Indicates stable demand; allows for "Just-in-Time" inventory.
  • Outlier Detection: Using MIN and MAX to identify "Hero Products" versus "Dead Stock" that is consuming liquid capital.

Practice: Outlier Detection

Which function would you use to find the single highest-performing "Hero Product" in the sales range B2:B100?

2. Logical Functions

  • IF: Returns one value if a condition is TRUE and another if it is FALSE.
    Example: If a sale is over ₹5,000, label it "Premium"; otherwise, "Standard."
  • AND: Returns TRUE only if all conditions are met.
    Example: Trigger a bonus only if Sales > ₹1,00,000 AND Attendance > 95%.
  • OR: Returns TRUE if at least one condition is met.
    Example: Flag an invoice for review if it is Overdue OR the amount exceeds ₹50,000.
  • NOT: Reverses the logic.
    Example: Identify all boutique branches that are NOT located in the "South" region.

Practice: Multi-Condition Logic

Which logical function should wrap around multiple conditions if a manager wants to flag an employee who meets every single one of the criteria?
