2.1 Defining AI Problems

Every AI journey begins with a clearly defined problem. Without a goal, a model is just an expensive random number generator.

Key Steps in Problem Definition

  • Identification: What is the core business or research challenge?
  • Data Availability: Do we have the right data to solve it?
  • Success Metrics: How will we know if the AI is "working"? (Accuracy, Precision, Recall)

2.2 Machine Learning Categories

Supervised Learning

The system is trained on a labeled dataset, meaning that it is provided with both the input data and the corresponding correct output.

Common Tasks: Spam detection, property price prediction, image classification.

Unsupervised Learning

The system is given unlabeled data and must find patterns or structures on its own.

Common Tasks: Customer segmentation, anomaly detection, topic modeling.

Reinforcement Learning

The system learns through trial and error by receiving rewards or penalties based on its actions.

Common Tasks: Game AI, robotics, stock market trading.
