1. Meaning of Data

Concept

Data refers to raw facts, figures, or observations collected for analysis or reference.

Data can appear in many forms such as:

  • Numbers
  • Text
  • Images
  • Audio

By itself, data does not carry meaning until it is processed and interpreted.

3. Data Variable Types

Concept

Variables describe the type of value stored in data. Main variable types include: Boolean, Numeric, String. Each type stores different kinds of values.

Insight: Understanding variable types helps store data correctly, analyze data efficiently, and avoid calculation errors.

3.1 Boolean Data Type

Boolean data type stores two possible values: True or False. Booleans are commonly used in logical conditions and decision making.

Example: is_logged_in = True
Boolean values are used in conditions like:

if is_logged_in:
    print("Access granted")

Numeric data types represent numbers that can be calculated mathematically. Examples: exam marks, salary, weight, height. Numeric types include Integer, Float, Complex.

3.3 Integer

Concept: An integer is a whole number without decimal values. Examples: 5, 10, -3, 0. Integers represent quantities like number of students or number of products sold.

Insight: Integers are useful when exact counts are needed.

3.4 Float

Concept: A float represents numbers with decimal values. Examples: 2.5, 3.14, 7.8. Float values are used when precision is required (e.g., price, temperature, weight).

Insight: Floats allow more accurate measurements compared to integers.

3.5 Complex Numbers

Concept: Complex numbers contain a real part and an imaginary part (e.g., 3 + 4i). Python supports complex numbers using the complex data type.

Insight: Complex numbers are mainly used in engineering and scientific calculations.

Concept: String data type stores text values (e.g., names, city, product names). Strings can contain letters, numbers, and symbols.

Example: name = "Anoop"
Insight: Strings help store descriptive information, not numeric calculations.

5. Types of Data

Data can be classified into two main types: Qualitative and Quantitative data.

5.1 Qualitative Data

Qualitative data describes categories or characteristics (e.g., gender, city, brand). Types include Nominal and Ordinal.

Insight: Qualitative data focuses on descriptions rather than numbers.

6. Structured vs Unstructured Data

Structured Data: Fits into rows and columns, usually stored in databases (e.g., phone numbers, bank records).

Unstructured Data: Has no fixed format (e.g., emails, videos, social media posts).

Insight: Most modern data generated online is unstructured.

7. Metadata

Concept: Metadata means data about data. It describes information such as file size, creation date, author, or format.

Example: A photo file may include metadata: camera type, date taken, resolution.
Insight: Metadata helps organize and manage data efficiently.

Concept: Big Data refers to extremely large and complex datasets that grow rapidly and cannot be processed using traditional systems.

Insight: Big data allows organizations to analyze massive amounts of information to discover patterns and predictions.
