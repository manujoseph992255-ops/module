Introduction to Python Modules

As programs grow larger, writing all code inside a single file becomes difficult to manage. Python solves this problem using modules.

A module is a file that contains reusable Python code such as:

  • Functions
  • Variables
  • Classes

Instead of writing everything manually, Python provides built-in modules that help solve common computing problems.

Examples of built-in modules:

  • math – mathematical calculations
  • random – random number generation
  • datetime – working with dates and time
  • os – operating system interaction
  • sys – system-level operations

Using modules makes programs: Faster to develop, More reliable, and Easier to maintain.

Importing Modules

To use a module, we must import it.

Example

import math
print(math.pi)

Output

3.141592653589793
math.pi retrieves the value of π.

Importing Specific Functions

Sometimes we only need a specific function from a module.

Example

from math import sqrt
print(sqrt(16))

Output

4.0
Explanation
Instead of importing the entire module, we import only the required function.

6.1 File System and Command-Line Operations

Some modules allow Python to interact with the computer system. These are useful when building real applications.

The os Module

The os module provides functions to interact with the operating system.

Get Current Working Directory Example:

import os
print(os.getcwd())

Output (example)

C:\Users\Student
getcwd() returns the current working directory where the program is running.

Coding Practice: OS Interaction

Challenge: Import os and print "Checking".

List Files in a Directory Example:

import os
print(os.listdir())

Output (example)

['report.txt', 'sales.py']
Explanation
listdir() displays files and folders in the current directory. This is useful for programs that process multiple data files.

Checking File Existence Example:

import os
print(os.path.exists("data.txt"))

Output

True
Explanation
This prevents FileNotFoundError before attempting to open a file.

The io Module

The io module provides advanced file input and output operations.

Example

import io
file = io.open("sample.txt", "w")
file.write("Financial Report")
file.close()
Explanation
io.open() behaves similarly to Python's built-in open() function. It is commonly used in advanced file handling systems.

The sys Module

The sys module provides access to system-level information.

Command-Line Arguments Example:

import sys
print(sys.argv)

If the script is executed as: python script.py Rahul 30

Output

['script.py', 'Rahul', '30']
Explanation
sys.argv stores command-line arguments in a list. First element → script name. Remaining elements → user inputs.

Python Version Information Example:

import sys
print(sys.version)

Output (example)

3.12.0 (main, ...)
Explanation
This shows the Python version currently running the program.

6.2 Solving Computing Problems Using Built-in Modules

Python modules provide many built-in tools for solving problems efficiently.

The math Module

The math module provides mathematical functions.

  • math.fabs(-10) → returns 10.0 (Absolute Value)
  • math.ceil(4.3) → returns 5 (Round Up)
  • math.floor(4.7) → returns 4 (Round Down)
  • math.trunc(4.9) → returns 4 (Truncation)
  • math.fmod(10, 3) → returns 1.0 (Modulus)
  • math.sqrt(16) → returns 4.0 (Square Root)
  • math.pow(2, 3) → returns 8.0 (Power)
  • math.isnan(math.nan) → returns True (Checking NaN-Not a Number)

The datetime Module

The datetime module is used to work with dates and time.

Current Date and Time Example:

from datetime import datetime
now = datetime.now()
print(now)

Output (example)

2026-03-02 14:30:45.123456
Explanation
datetime.now() returns the current system date and time.

Formatting Dates Example:

from datetime import datetime
now = datetime.now()
print(now.strftime("%Y-%m-%d"))

Output

2026-03-02
Explanation
strftime() converts the date into a formatted string.

Finding the Day of the Week Example:

from datetime import datetime
print(datetime.now().weekday())

Output (example)

0
Explanation
Days are numbered: 0 → Monday, 6 → Sunday.

The random Module

The random module generates random values. This is useful for simulations, testing, and sampling.

  • random.randint(1, 5) → Random integer between 1-5
  • random.random() → Random decimal between 0 and 1
  • random.choice(["A", "B", "C"]) → Random selection from a list
  • random.shuffle(numbers) → Randomly rearranges list elements
  • random.sample(list, 2) → Selects random elements without repetition

Integrated Business Example

Example: Financial data simulation.

import math
import random
from datetime import datetime

revenue = random.randint(10000, 50000)
growth = math.sqrt(revenue)
today = datetime.now().strftime("%Y-%m-%d")

print("Revenue:", revenue)
print("Growth Indicator:", growth)
print("Report Date:", today)

Sample Output

Revenue: 25000
Growth Indicator: 158.11
Report Date: 2026-03-05
Explanation
This example demonstrates: Random revenue generation, Mathematical analysis, and Date stamping for reports.
