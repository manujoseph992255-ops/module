Introduction to Flow Control

In Module 1, we learned:

  • Data types
  • Operators
  • Boolean expressions

Now we use those concepts to control the execution of a program.

Flow control allows a program to:

  • Make decisions
  • Repeat actions
  • Process data automatically

For example, in business applications Python can be used to:

  • Check whether a company made profit or loss
  • Categorize sales performance
  • Process multiple transactions
  • Analyze financial records

Flow control is implemented using:

  • Decision statements (if)
  • Loops (while, for)

2.1 Branching Statements (Decision Making)

Branching allows a program to choose different actions depending on conditions.

if Statement

The if statement is the most basic form of decision making. It allows you to run a block of code only if a specific condition is True. If the condition is False, the code is skipped.

Purpose

Use the if statement when you want to perform an action based on a single condition, such as verifying a user's age or checking if a stock is available.

Example:

profit = 5000
if profit > 0:
    print("The business made a profit")

Output

The business made a profit

Coding Practice: Basic if

Challenge: Create a variable called price and assign a value to it. Write a program that checks if the value of price is greater than 100. If the price is greater than 100, print "Expensive".

if – else Statement

The if-else statement provides two paths for the program. One path executes if the condition is True, and the other path (the else block) executes if the condition is False.

Purpose

Use if-else when you have two mutually exclusive outcomes, like checking if a transaction is "Approved" or "Declined".

Example

profit = -2000
if profit > 0:
    print("Profit")
else:
    print("Loss")

Output

Loss

Coding Practice: if-else

Challenge: Write a program that checks if score is 50 or more. If True, print "Pass", otherwise print "Fail".

if – elif – else Statement

The elif (short for else if) statement allows you to check multiple conditions one after another. Only the first condition that is True will be executed.

Purpose

Use if-elif-else for categorization problems, such as grading students (A, B, C, D) or determining tax brackets based on income.

Example

sales = 8500
if sales >= 10000:
    print("Excellent Sales")
elif sales >= 7000:
    print("Good Sales")
elif sales >= 4000:
    print("Average Sales")
else:
    print("Low Sales")

Output

Good Sales

Nested if Statements

A nested if is simply an if statement placed inside another if. This allows for hierarchical or multi-level decision making.

Purpose

Use Nested if when a decision depends on a previous decision. For example, check if a user is logged in, and THEN check if they have admin permissions.

Example

profit = 8000
if profit > 0:
    if profit > 5000:
        print("High Profit")
    else:
        print("Moderate Profit")
else:
    print("Business Loss")

Output

High Profit

Coding Practice: Nested if

Compound Conditional Expressions

Multiple conditions can be combined using logical operators.

Logical operators:

  • and
  • or
  • not

Example

age = 25
has_account = True
if age >= 18 and has_account:
    print("Customer eligible for loan")

Output

Customer eligible for loan
Both conditions must be True for the statement to execute.

2.2 Iteration (Loops)

Loops allow a program to repeat a block of code multiple times.

Loops are useful for processing:

  • Sales records
  • Transactions
  • Customer lists
  • Financial data

Python provides two main loops:

  • while loop
  • for loop

while Loop

The while loop repeats a block of code as long as a specified condition is True. It is often used when you don't know exactly how many times you need to repeat something.

Purpose

Use while loops for tasks like monitoring a system until an event occurs (e.g., waiting for a file to upload) or processing data until a "Stop" signal is received.

Example

day = 1
while day <= 3:
    print("Processing sales data for day", day)
    day += 1

Output

Processing sales data for day 1
Processing sales data for day 2
Processing sales data for day 3

Coding Practice: while Loop

Challenge: Fix the loop to print numbers from 1 to 5.

for Loop

The for loop is used to iterate over a sequence (like a list, tuple, or range). It is best used when you know the number of times you want to repeat a block of code.

Purpose

Use for loops for processing every item in a list, such as calculating total revenue from a list of transactions or sending an email to every customer in a database.

Example

for i in range(3):
    print("Monthly sales report", i)

Output

Monthly sales report 0
Monthly sales report 1
Monthly sales report 2

Loop with Conditional Logic

Loops can also contain decision statements.

Example

sales = [5000, 9000, 6500, 12000]
for s in sales:
    if s >= 8000:
        print("High sales:", s)

Output

High sales: 9000
High sales: 12000
Explanation
The loop checks each sales value and prints only those greater than or equal to 8000.

break Statement

The break statement is used to exit a loop prematurely, even if the loop condition is still True.

Purpose

Use break to stop searching once you've found what you're looking for, or to terminate a program if a critical error occurs during processing.

Example

for i in range(5):
    if i == 3:
        break
    print(i)

Output

0, 1, 2

Coding Practice: break

Challenge: Write a loop that prints numbers from 1 to 10, but use break to stop the loop when the number reaches 4.

continue Statement

The continue statement skips the current block of code and moves directly to the next iteration of the loop.

Purpose

Use continue to skip specific items that don't meet a criteria, such as ignoring "Cancelled" orders in a list while processing the "Active" ones.

Example

for i in range(5):
    if i == 2:
        continue
    print(i)

Output

0, 1, 3, 4

pass Statement

The pass statement is a null operation. It is used as a placeholder when a statement is syntactically required but no action is needed.

Purpose

Use pass when you are sketching out a program structure and plan to write the logic later, or when creating a minimal class or function.

Nested Loops

A nested loop is a loop inside another loop.

Example

for month in range(2):
    for week in range(3):
        print("Month:", month, "Week:", week)

Output

Month: 0 Week: 0
Month: 0 Week: 1
Month: 0 Week: 2
Month: 1 Week: 0
Month: 1 Week: 1
Month: 1 Week: 2
Explanation
The inner loop runs completely for each iteration of the outer loop.

Integrated Business Example

Example: Sales performance analysis.

sales = [6500, 8800, 7200, 9000, 5400]
for s in sales:
    if s >= 8000:
        print(s, "High Sales")
    else:
        print(s, "Average Sales")

Output

6500 Average Sales
8800 High Sales
7200 Average Sales
9000 High Sales
5400 Average Sales
Explanation
This example combines:
• Lists
• Loops
• Conditional statements
to analyze sales performance.
