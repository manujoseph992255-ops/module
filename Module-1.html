1.1 Understanding Data Types

Python automatically assigns a data type based on the value stored.

This is called dynamic typing.

Integer (int)

Whole numbers without decimals.

Example:

units_sold = 150
print(type(units_sold))

Output

<class 'int'>
The value 150 is a whole number, so Python assigns it the integer data type.

Coding Practice: Integer

Challenge: Assign the number 250 to a variable called stock and print its type.

Float (float)

Numbers with decimal values.

Example

product_price = 199.99
print(type(product_price))

Output

<class 'float'>
Decimal numbers are stored as float values.

Coding Practice: Float

Challenge: Create a variable discount with value 0.15 and print its type.

String (str)

Strings store text data such as names, product labels, or company names.

Example

company_name = "Global Finance Ltd"
print(type(company_name))

Output

<class 'str'>

Coding Practice: String

Challenge: Store the name "KVJ" in a variable and print its type.

Important comparison

print(type("10"))
print(type(10))

Output

<class 'str'>
<class 'int'>
Explanation
"10" → text
10 → number

Boolean (bool)

Boolean values represent True or False.

Example

profit = 5000
result = profit > 0
print(result)
print(type(result))

Output

True
<class 'bool'>
The comparison profit > 0 returns a Boolean value.

Coding Practice: Boolean

Challenge: Write an expression that checks if 500 > 200 and print the result.

Evaluating Expressions

sales = 5000
tax_rate = 0.05
customer = "Rahul"
status = sales > 3000
print(type(sales))
print(type(tax_rate))
print(type(customer))
print(type(status))

Output

<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>
Explanation
sales → integer
tax_rate → float
customer → string
status → boolean

Type Conversion

Type conversion changes data from one type to another.

Example

x = int("100")
y = float("10.5")
z = str(500)
w = bool(0)
print(x)
print(y)
print(z)
print(w)

Output

100
10.5
500
False
Explanation
bool() converts to Boolean

1.2 Constructing Data Structures

Python provides structures for storing collections of data.

List

A list stores ordered values.

Example: monthly sales data.

monthly_sales = [5000, 7200, 6500]
print(monthly_sales)

Output

[5000, 7200, 6500]

Characteristics

  • Ordered
  • Mutable
  • Allows duplicates

Indexing

print(monthly_sales[0])
print(monthly_sales[-1])

Output

5000
6500
Index 0 refers to the first value.

Coding Practice: Indexing

Challenge: From the list prices = [10.5, 20.0, 35.5], print the **second** element.

Coding Practice: List Indexing

Challenge: Create a list called scores with values [85, 90, 95] and print the first element.

Slicing

print(monthly_sales[0:2])
print(monthly_sales[:2])
print(monthly_sales[1:])

Output

[5000, 7200]
[5000, 7200]
[7200, 6500]

List Operations

append()

monthly_sales.append(9000)
print(monthly_sales)

Output

[5000, 7200, 6500, 9000]

insert()

monthly_sales.insert(1, 6000)
print(monthly_sales)

Output

[5000, 6000, 7200, 6500, 9000]

remove()

monthly_sales.remove(6500)
print(monthly_sales)

sort()

monthly_sales.sort()
print(monthly_sales)

reverse()

monthly_sales.reverse()
print(monthly_sales)

max() and min()

print(max(monthly_sales))
print(min(monthly_sales))

Tuple

Tuples store fixed data that should not change.

Example: store coordinates of a branch office.

location = (10, 20)
print(location)

Tuples are immutable.

Set

Sets store unique values.

Example: unique product IDs.

product_ids = {101, 102, 103, 103}
print(product_ids)

Output

{101, 102, 103}

Dictionary

Dictionaries store key-value pairs.

Example: customer information.

customer = {
    "name": "Rahul",
    "balance": 5000,
    "account_type": "Savings"
}
print(customer["name"])

1.3 Operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations. Here is the complete list of operators in Python:

  • Addition (+): Adds two numbers. (10 + 5 = 15)
  • Subtraction (-): Subtracts one number from another. (10 - 5 = 5)
  • Multiplication (*): Multiplies two numbers. (10 * 5 = 50)
  • Division (/): Divides numbers and returns a decimal (float). (10 / 4 = 2.5)
  • Floor Division (//): Divides and rounds down to the nearest whole number. (10 // 3 = 3)
  • Modulus (%): Returns the remainder of a division. (10 % 3 = 1)
  • Exponentiation (**): Raises a number to a power. (2 ** 3 = 8)

Let's practice the most common ones!

Comparison operators

print(10 > 5)
print(10 == 5)
print(10 != 5)

Logical operators

print(True and False)
print(True or False)
print(not True)

Assignment operators

revenue = 10000
revenue += 2000
print(revenue)

Identity operator

a = 10
b = 10
print(a is b)

Membership operator

products = ["Laptop", "Phone", "Tablet"]
print("Phone" in products)

1.4 Operator Precedence

result = 10 + 5 * 2
print(result)

Output

20

Multiplication happens before addition.

Using parentheses

print((10 + 5) * 2)

Output

30
